# Шаблон для теста редактора

В репозитории **нет бинарных файлов**. Поэтому `template.docx` нужно добавить вручную.

## Что сделать
1. Создайте файл `app/templates/template.docx`.
2. Добавьте в него:
   - заголовок H1,
   - маркированный список,
   - нумерованный список,
   - таблицу 2x2.
3. Откройте AI-диалог и проверьте, что структура корректно отображается в HTML-редакторе и после экспорта.

Подробный чеклист: `app/templates/template-checklist.md`.
